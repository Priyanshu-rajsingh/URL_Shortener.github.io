Test Cases

Below are the test cases to verify the functionality of the URL shortener.

# Scenario Input Expected Output
1 Valid Long URL https://www.google.com Shortened URL is displayed and redirects correctly.
2 Invalid URL invalid-url Error message: "Please enter a valid URL."
3 Empty Input (Empty field) Error message: "Please enter a valid URL."
4 Very Long URL https://example.com/... (long) Shortened URL is displayed and redirects correctly.
5 Invalid API Key Replace API key with invalid one Error message: "Failed to shorten URL. Please try again."
6 URL Without HTTP/HTTPS www.example.com Shortened URL is displayed and works as expected.
7 Special Characters in URL https://example.com?name=John+Doe Shortened URL works and redirects correctly.
8 Nonexistent URL https://nonexistent.com Shortened URL is created; clicking may not resolve.
9 High Traffic Load Submit multiple URLs rapidly Each URL is shortened without errors.
10 Shorten Shortened URL TinyURL or Bitly URL New shortened URL is generated.
